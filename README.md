## Функциональность проекта

### Реализованные возможности:

#### 1. Добавление записи о студенте
- Ввод данных: ID, имя, фамилия, возраст, пол
- **Проверки:**
  - ID должен быть уникальным (нельзя создать две записи с одинаковым ID)
  - Возраст не может быть отрицательным
  - Все поля обязательны для заполнения

#### 2. Просмотр всех записей
- Вывод полного списка всех студентов в базе данных

#### 3. Поиск записей по фильтрам
- Гибкий поиск по любому полю (ID, имя, фамилия, возраст, пол)
- Если поле не указано (оставлено пустым) — оно не участвует в фильтрации
- Возможность комбинировать несколько фильтров одновременно

#### 4. Обработка ошибок
- Защита от некорректного ввода (возраст должен быть числом)
- Информативные сообщения об ошибках
- Предотвращение дублирования записей
